## Install

```bash
dune theme:install jsr:@dune/theme-stellar@1.0.0 --activate
```

Or by hand, in `config/site.yaml`:

```yaml
themes:
  - name: stellar
    src: jsr:@dune/theme-stellar@1.0.0

theme:
  name: stellar
  src: jsr:@dune/theme-stellar@1.0.0
```

Multipurpose landing with logo header, spotlight, and statistics. See the
[live demo](https://themes.getdune.org/stellar).

## Templates

| Template | Role |
|---|---|
| `default` | Spotlight/features landing on home; main content elsewhere |
| `blog` | Post listing |
| `post` | Single post |
| `archives` | Year-grouped post list |
| `search` | Form search |
| `error` | Themed 404 / 500 |
