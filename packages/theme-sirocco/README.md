# Sirocco

Original Dune theme, design inspired by [hugo-PaperMod](https://github.com/adityatelange/hugo-PaperMod) — a fast, minimalist blog theme.

**Templates**: `default` (plain page) · `post` (date, reading time, author, tags) · `blog` (card list with pagination, also works as home) · `archives` · `search` · `error`

**Config schema** (admin-editable): `accent_color`, `default_dark`, `show_reading_time`, `home_subtitle`

Light/dark toggle persisted in localStorage; scroll-to-top button; tag links point at `/tag:<name>` taxonomy routes. UI strings live in `locales/en.json`.
