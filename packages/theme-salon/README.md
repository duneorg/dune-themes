# Salon

Original Dune theme, design inspired by [Liebling](https://github.com/eddiesigner/liebling) — magazine-style blog.

**Templates**: `default` · `post` · `blog` · `search` · `error`

**Config schema**: `accent_color`, `default_dark`, `home_subtitle`, `footer_text`

Featured lead post on the blog index, card grid for remaining posts, cover images and category tags. Post frontmatter: `cover`, `tags` or `taxonomy.tag`.

```bash
dune theme:install jsr:@dune/theme-salon@1.0.0 --activate
```

Not a port of Liebling.
