## Install

```bash
dune theme:install jsr:@dune/theme-arcana@1.0.0 --activate
```

Or by hand, in `config/site.yaml`:

```yaml
themes:
  - name: arcana
    src: jsr:@dune/theme-arcana@1.0.0

theme:
  name: arcana
  src: jsr:@dune/theme-arcana@1.0.0
```

Logo header, optional home banner, and wrapper pages. See the
[live demo](https://themes.getdune.org/arcana).
