## Install

```bash
dune theme:install jsr:@dune/theme-halcyonic@1.0.0 --activate
```

Or by hand, in `config/site.yaml`:

```yaml
themes:
  - name: halcyonic
    src: jsr:@dune/theme-halcyonic@1.0.0

theme:
  name: halcyonic
  src: jsr:@dune/theme-halcyonic@1.0.0
```

Business landing with banner and feature grid. See the
[live demo](https://themes.getdune.org/halcyonic).

## Templates

| Template | Role |
|---|---|
| `default` | Banner + features on home; content pages elsewhere |
| `blog` | Post listing |
| `post` | Single post |
| `archives` | Year-grouped post list |
| `search` | Form search |
| `error` | Themed 404 / 500 |
