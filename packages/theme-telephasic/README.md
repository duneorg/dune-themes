# Telephasic

Dune theme adapted from [HTML5 UP Telephasic](https://html5up.net/telephasic) by [HTML5 UP](https://html5up.net).

**License:** Design by HTML5 UP (CC BY 3.0). Keep visible design credit on live sites.

**Templates:** `default` · `post` · `blog` · `search` · `error` · `archives` — business landing with hero, feature rows, promo band, and no-sidebar inner pages.

```bash
dune theme:install jsr:@dune/theme-telephasic@1.0.0 --activate
```
