## Install

```bash
dune theme:install jsr:@dune/theme-parallelism@1.0.0 --activate
```

Or by hand, in `config/site.yaml`:

```yaml
themes:
  - name: parallelism
    src: jsr:@dune/theme-parallelism@1.0.0

theme:
  name: parallelism
  src: jsr:@dune/theme-parallelism@1.0.0
```

Horizontal masonry on home and blog, wheel/keyboard scroll, and edge
scroll zones. See the [live demo](https://themes.getdune.org/parallelism).

## Templates

| Template | Role |
|---|---|
| `default` | Masonry gallery landing on home; prose pages elsewhere |
| `blog` | Masonry thumb post listing |
| `post` | Single post |
| `archives` | Year-grouped post list |
| `search` | Form search |
| `error` | Themed 404 / 500 |
