# Tessellate

Dune theme adapted from [HTML5 UP Tessellate](https://html5up.net/tessellate) by [HTML5 UP](https://html5up.net).

**License:** Design by HTML5 UP (CC BY 3.0). Keep visible design credit on live sites.

**Templates:** `default` · `post` · `blog` · `search` · `error` · `archives` — scroll-section landing with icon features, image grids, and dark content bands.

```bash
dune theme:install jsr:@dune/theme-tessellate@1.0.0 --activate
```
