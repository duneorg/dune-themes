# Oasis

Original Dune theme, design inspired by [Hugo Blox](https://hugoblox.com) (Tailwind) — landing pages and academic résumé sites.

**Templates**: `default` · `landing` (avatar hero with role/organization/social links + biography + collection cards) · `blog` (card grid) · `post`

**Config schema**: `primary_color`, `avatar_url`, `role`, `organization`

Hero fields can also be set per-page under `hero:` frontmatter. Tailwind-style visual language: Inter, soft shadows, rounded cards, dark mode toggle.
