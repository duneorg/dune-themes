# Ink

Original Dune theme, design inspired by [Attila](https://github.com/zutrinken/attila) — long-form serif writing.

**Templates**: `default` · `post` (reading time, author box) · `blog` · `search` · `error`

**Config schema**: `accent_color`, `default_dark`, `show_reading_time`, `home_subtitle`, `author_name`, `author_bio`, `author_avatar`, `twitter_url`, `github_url`, `footer_text`

Centered masthead, dark mode toggle, tag links at `/tag:<name>`.

```bash
dune theme:install jsr:@dune/theme-ink@0.1.0 --activate
```

Not a port of Attila. For a near-identical upstream experience, use a faithful port from the [dune-themes](https://github.com/duneorg/dune-themes) catalog when available.
