# Manual

Original Dune theme, design inspired by [Just the Docs](https://github.com/just-the-docs/just-the-docs) — product documentation.

**Templates**: `default` · `section` · `post` · `search` · `error`

**Config schema**: `accent_color`, `default_dark`, `site_tagline`, `footer_text`

Full-height sidebar with search link, active nav accent border. Page frontmatter `label: New` or `Beta` renders a badge on the title.

```bash
dune theme:install jsr:@dune/theme-manual@0.1.0 --activate
```

Not a port of Just the Docs. For book-style sidebar docs, see **Caravan**.
