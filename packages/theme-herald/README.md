# Herald

Original Dune theme, design inspired by [Casper](https://github.com/TryGhost/Casper) — publication-style blog.

**Templates**: `default` · `post` · `blog` · `search` · `error` (home hero is in the layout)

**Config schema**: `accent_color`, `default_dark`, `hero_title`, `hero_subtitle`, `hero_image`, `default_author`, `default_avatar`, `footer_text`

Dark masthead, full-width home hero with optional background image, excerpt feed with author avatars and cover thumbnails.

```bash
dune theme:install jsr:@dune/theme-herald@0.1.0 --activate
```

Not a port of Casper.
