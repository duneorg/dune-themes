## `color_scheme`

One of eight curated presets. Default **purple** uses a Starlight-adjacent
violet with matched light/dark surfaces. This demo turns on the visitor
scheme switcher.

## `default_dark`

On by default for Nightfall. Header toggle still switches at runtime; with
no stored choice, dark wins (or OS preference if you turn this off).

## `github_url`

Optional header icon linking to your repository.

## `tagline`

Used by the splash template hero when `hero.tagline` is omitted.
