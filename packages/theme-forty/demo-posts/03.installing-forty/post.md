taxonomy:
  tag: [install, forty]
---

```bash
dune theme:install jsr:@dune/theme-forty@1.0.0 --activate
```

Or extract `forty-1.0.0.zip` into `themes/forty/` and set `theme.name: forty`
in `site.yaml`.
