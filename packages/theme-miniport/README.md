## Install

```bash
dune theme:install jsr:@dune/theme-miniport@1.0.0 --activate
```

Or by hand, in `config/site.yaml`:

```yaml
themes:
  - name: miniport
    src: jsr:@dune/theme-miniport@1.0.0

theme:
  name: miniport
  src: jsr:@dune/theme-miniport@1.0.0
```

Sticky top nav, portrait hero, and icon-box explore section. See the
[live demo](https://themes.getdune.org/miniport).

## Templates

| Template | Role |
|---|---|
| `default` | Portrait hero + icon boxes on home; plain pages elsewhere |
| `blog` | Icon-box post listing |
| `post` | Single post |
| `archives` | Year-grouped post list |
| `search` | Form search |
| `error` | Themed 404 / 500 |
