## Install

```bash
dune theme:install jsr:@dune/theme-dimension@1.0.0 --activate
```

Or by hand, in `config/site.yaml`:

```yaml
themes:
  - name: dimension
    src: jsr:@dune/theme-dimension@1.0.0

theme:
  name: dimension
  src: jsr:@dune/theme-dimension@1.0.0
```

Full-screen landing with modal article panels. See the
[live demo](https://themes.getdune.org/dimension).

## Templates

| Template | Role |
|---|---|
| `default` | Landing header on home; modal article panels elsewhere |
| `blog` | Post listing |
| `post` | Single post |
| `archives` | Year-grouped post list |
| `search` | Form search |
| `error` | Themed 404 / 500 |
