## `color_scheme`

One of eight curated presets. Default **terracotta** uses a warm orange
(`#ea580c`) in the Astrofy family, with matched light/dark surfaces. This
demo turns on the visitor scheme switcher.

## `scheme_switcher`

On in this demo. Swatch dropdown next to the dark-mode toggle.

## `avatar_url`

Optional sidebar portrait. This demo uses a local SVG under the theme
static folder.

## `hero_greeting`

Shown above the home title. Default is "Hey there" (no emoji).

## `footer_text`

Optional override for the footer line.
