## Install

```bash
dune theme:install jsr:@dune/theme-lens@1.0.0 --activate
```

Or by hand, in `config/site.yaml`:

```yaml
themes:
  - name: lens
    src: jsr:@dune/theme-lens@1.0.0

theme:
  name: lens
  src: jsr:@dune/theme-lens@1.0.0
```

Staggered preload, icon nav, and thumbnail gallery on home and blog. See
the [live demo](https://themes.getdune.org/lens).

## Templates

| Template | Role |
|---|---|
| `default` | Thumbnail gallery on home; content panel elsewhere |
| `blog` | Thumbnail post listing |
| `post` | Single post |
| `archives` | Year-grouped post list |
| `search` | Form search |
| `error` | Themed 404 / 500 |
