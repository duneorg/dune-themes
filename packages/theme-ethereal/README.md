## Install

```bash
dune theme:install jsr:@dune/theme-ethereal@1.0.0 --activate
```

Or by hand, in `config/site.yaml`:

```yaml
themes:
  - name: ethereal
    src: jsr:@dune/theme-ethereal@1.0.0

theme:
  name: ethereal
  src: jsr:@dune/theme-ethereal@1.0.0
```

Banner panel, spotlights, and icon grid on home. See the
[live demo](https://themes.getdune.org/ethereal).

## Templates

| Template | Role |
|---|---|
| `default` | Panel landing on home; plain pages elsewhere |
| `blog` | Post listing |
| `post` | Single post |
| `archives` | Year-grouped post list |
| `search` | Form search |
| `error` | Themed 404 / 500 |
