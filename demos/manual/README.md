# Manual demo site

Local preview of `@dune/theme-manual` using shared docs fixture content.
The theme README is the homepage (same pattern as caravan/book/lucid).

```bash
# From repo root
deno task demo manual
deno task demo:validate manual
```

Open http://localhost:8821 after `demo` starts.

**Deploy target:** https://themes.getdune.org/manual

`content/` is generated by `deno task demo:link manual`. Theme is
symlinked from `packages/theme-manual/`.
