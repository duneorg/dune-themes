# Herald demo site

Local preview of `@dune/theme-herald` using shared blog fixture content.

```bash
# From repo root
deno task demo herald
deno task demo:validate herald
```

Open http://localhost:8819 after `demo` starts.

**Deploy target:** https://themes.getdune.org/herald

`content/` is generated by `deno task demo:link herald` — copied from
`demos/_shared/blog/` (re-run `demo:link` after editing the source; it
isn't live-linked). Theme is symlinked from `packages/theme-herald/`.
