# Lucid demo site

Local preview of `@dune/theme-lucid` using shared docs fixture content.
The theme README is the homepage (same pattern as caravan/book/starlight).

```bash
# From repo root
deno task demo lucid
deno task demo:validate lucid
```

Open http://localhost:8820 after `demo` starts.

**Deploy target:** https://themes.getdune.org/lucid

`content/` is generated by `deno task demo:link lucid`. Theme is
symlinked from `packages/theme-lucid/`.
