# Ink demo site

Local preview of `@dune/theme-ink` using shared blog fixture content.

```bash
# From repo root
deno task demo ink
deno task demo:validate ink
```

Open http://localhost:8815 after `demo` starts.

**Deploy target:** https://themes.getdune.org/ink

`content/` is generated by `deno task demo:link ink` — copied from
`demos/_shared/blog/` (re-run `demo:link` after editing the source; it
isn't live-linked). Theme is symlinked from `packages/theme-ink/`.
