# PaperMod demo site

Local preview of `@dune/theme-papermod` using shared blog fixture content.

```bash
# From repo root
deno task demo papermod
deno task demo:validate papermod
```

Open http://localhost:8765 after `demo` starts.

**Deploy target:** https://themes.getdune.org/papermod

`content/` is generated by `deno task demo:link papermod` — copied from
`demos/_shared/blog/` plus an auto-generated "About" page rendered from
`packages/theme-papermod/README.md` (re-run `demo:link` after editing either
source; it isn't live-linked). Theme is symlinked from
`packages/theme-papermod/`.
