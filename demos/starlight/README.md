# Starlight demo site

Local preview of `@dune/theme-starlight` using the shared docs fixture content.

```bash
# From repo root
deno task demo starlight
deno task demo:validate starlight
```

Open http://localhost:8813 after `demo` starts.

**Deploy target:** https://themes.getdune.org/starlight

`content/` is generated by `deno task demo:link starlight` — copied from
`demos/_shared/docs/`, with the homepage (`01.home`) replaced by
`packages/theme-starlight/README.md` (docs themes opt into this via
`DEMO_README_AS_HOME`). Theme-specific docs fold in at `02.theme-docs/`
and the style guide at `06.style-guide/`. Re-run `demo:link` after editing
either source. Theme is symlinked from `packages/theme-starlight/`.
