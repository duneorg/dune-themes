# Caravan demo site

Local preview of `@dune/theme-caravan` using the shared docs fixture content.

```bash
# From repo root
deno task demo caravan
deno task demo:validate caravan
```

Open http://localhost:8810 after `demo` starts.

**Deploy target:** https://themes.getdune.org/caravan

`content/` is generated by `deno task demo:link caravan` — copied from
`demos/_shared/docs/`, with the homepage (`01.home`) replaced by
`packages/theme-caravan/README.md` (docs themes are opted into this via
`DEMO_README_AS_HOME` in `scripts/demo-common.ts` — most demos get the
README as a separate `/about-theme/` page instead, since replacing their
actual homepage would hide the thing being demonstrated). Re-run
`demo:link` after editing either source; neither is live-linked. Theme is
symlinked from `packages/theme-caravan/`.
