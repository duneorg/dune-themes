# Caravan demo site

Local preview of `@dune/theme-caravan` using the shared docs fixture content.

```bash
# From repo root
deno task demo caravan
deno task demo:validate caravan
```

Open http://localhost:8810 after `demo` starts.

**Deploy target:** https://themes.getdune.org/caravan

`content/` is generated by `deno task demo:link caravan` — copied from
`demos/_shared/docs/` plus an auto-generated "About" page rendered from
`packages/theme-caravan/README.md` (re-run `demo:link` after editing either
source; it isn't live-linked). Theme is symlinked from
`packages/theme-caravan/`.
