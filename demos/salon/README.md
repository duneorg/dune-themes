# Salon demo site

Local preview of `@dune/theme-salon` using shared blog fixture content.

```bash
# From repo root
deno task demo salon
deno task demo:validate salon
```

Open http://localhost:8817 after `demo` starts.

**Deploy target:** https://themes.getdune.org/salon

`content/` is generated by `deno task demo:link salon` — copied from
`demos/_shared/blog/` (re-run `demo:link` after editing the source; it
isn't live-linked). Theme is symlinked from `packages/theme-salon/`.
