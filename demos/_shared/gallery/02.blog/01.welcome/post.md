This demo site showcases a portfolio/gallery Dune theme — a designed home
page, project listing, and thin inner pages so CTA destinations stay live.

## What to try

- Start on the **home** hero / gallery layout
- Open **Projects** (Blog) for the work listing
- Visit **About**, **Search**, and **Archives** from the menu or homepage tiles
- Trigger a **404** with a made-up URL to check the themed error page
