# About this demo

This site is part of the [dune-themes](https://github.com/duneorg/dune-themes)
monorepo. It uses fixture content shared across blog-style theme demos.

Install the theme on your own site:

```bash
dune theme:install jsr:@dune/theme-papermod@1.0.0 --activate
```

Or extract the release ZIP into `themes/papermod/`.
