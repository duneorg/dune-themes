## Caravan

Documentation theme inspired by [hugo-book](https://github.com/alex-shpak/hugo-book) — sticky sidebar nav, live search, dark mode. See its own README/demo for the full feature and config rundown.

```bash
dune theme:install jsr:@dune/theme-caravan@1.1.0 --activate
```

Demo: https://themes.getdune.org/caravan
