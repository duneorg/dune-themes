This is a shared documentation-site fixture used by Dune's docs-family
themes (Caravan, and later Book/Lucid/Manual). It exercises the pieces a
docs theme needs to prove itself: a sidebar built from top-level sections,
nested pages several levels deep, in-page markdown (code blocks, tables,
blockquotes), a section index page listing its children, search, and a
missing-page (404) case.

## What's in this fixture

- **Getting Started** — a shallow, linear onboarding flow (install, configure, first site)
- **Guides** — task-oriented docs (content model, theming, deployment)
- **Reference** — lookup-style docs (CLI commands, configuration keys)

Try the sidebar search for "install", "collection", or "cli". Try shrinking
the browser to see the mobile drawer. Try `/nonexistent-page` for the 404.
