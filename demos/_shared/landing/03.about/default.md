This site is part of the [dune-themes](https://github.com/duneorg/dune-themes)
monorepo. It uses fixture content shared across landing-style theme demos —
a designed homepage plus thin inner pages so CTA destinations stay live.

Install a theme on your own site:

```bash
dune theme:install jsr:@dune/theme-<slug>@1.0.0 --activate
```

Or extract the release ZIP into `themes/<slug>/`.
