# Fennec demo site

Local preview of `@dune/theme-fennec` using the shared portfolio fixture
plus a Fennec-specific home override.

```bash
# From repo root
deno task demo fennec
deno task demo:validate fennec
```

Open http://localhost:8822 after `demo` starts.

**Deploy target:** https://themes.getdune.org/fennec

`content/` is generated by `deno task demo:link fennec`. Theme is
symlinked from `packages/theme-fennec/`.
