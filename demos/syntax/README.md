# Syntax demo site

Local preview of `@dune/theme-syntax` using shared blog fixture content.

```bash
# From repo root
deno task demo syntax
deno task demo:validate syntax
```

Open http://localhost:8818 after `demo` starts.

**Deploy target:** https://themes.getdune.org/syntax

`content/` is generated by `deno task demo:link syntax` — copied from
`demos/_shared/blog/` (re-run `demo:link` after editing the source; it
isn't live-linked). Theme is symlinked from `packages/theme-syntax/`.
