# Nightfall demo site

Local preview of `@dune/theme-nightfall` using shared docs fixture content.
The theme README is the homepage (same pattern as caravan/lucid/manual).

```bash
# From repo root
deno task demo nightfall
deno task demo:validate nightfall
```

Open http://localhost:8823 after `demo` starts.

**Deploy target:** https://themes.getdune.org/nightfall

`content/` is generated by `deno task demo:link nightfall`. Theme is
symlinked from `packages/theme-nightfall/`.
