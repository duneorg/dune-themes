# Book demo site

Local preview of `@dune/theme-book` using the shared docs fixture content.

```bash
# From repo root
deno task demo book
deno task demo:validate book
```

Open http://localhost:8811 after `demo` starts.

**Deploy target:** https://themes.getdune.org/book

`content/` is generated by `deno task demo:link book` — copied from
`demos/_shared/docs/`, with the homepage (`01.home`) replaced by
`packages/theme-book/README.md` (docs themes are opted into this via
`DEMO_README_AS_HOME` in `scripts/demo-common.ts`). Theme-specific docs
fold in at `02.theme-docs/` and the style guide at `06.style-guide/`.
Re-run `demo:link` after editing either source; neither is live-linked.
Theme is symlinked from `packages/theme-book/`.
