# Gale demo site

Local preview of `@dune/theme-gale` using shared blog fixture content.

```bash
# From repo root
deno task demo gale
deno task demo:validate gale
```

Open http://localhost:8816 after `demo` starts.

**Deploy target:** https://themes.getdune.org/gale

`content/` is generated by `deno task demo:link gale` — copied from
`demos/_shared/blog/` (re-run `demo:link` after editing the source; it
isn't live-linked). Theme is symlinked from `packages/theme-gale/`.
