# Oasis demo site

Local preview of `@dune/theme-oasis` using shared portfolio fixture content.
Homepage is the academic CV landing (`demo-home`), not the theme README.

```bash
# From repo root
deno task demo oasis
deno task demo:validate oasis
```

Open http://localhost:8824 after `demo` starts.

**Deploy target:** https://themes.getdune.org/oasis

`content/` is generated by `deno task demo:link oasis`. Theme is
symlinked from `packages/theme-oasis/`.
