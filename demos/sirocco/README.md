# Sirocco demo site

Local preview of `@dune/theme-sirocco` using shared blog fixture content.

```bash
# From repo root
deno task demo sirocco
deno task demo:validate sirocco
```

Open http://localhost:8811 after `demo` starts.

**Deploy target:** https://themes.getdune.org/sirocco

`content/` is generated by `deno task demo:link sirocco` — copied from
`demos/_shared/blog/` plus an auto-generated "About" page rendered from
`packages/theme-sirocco/README.md` (re-run `demo:link` after editing either
source; it isn't live-linked). Theme is symlinked from
`packages/theme-sirocco/`.
