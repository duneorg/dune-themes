# Blox demo site

Local preview of `@dune/theme-blox` using the shared portfolio fixture plus
a theme-specific Academic CV landing homepage.

```bash
# From repo root
deno task demo blox
deno task demo:validate blox
```

Open http://localhost:8814 after `demo` starts.

**Deploy target:** https://themes.getdune.org/blox

`content/` is generated by `deno task demo:link blox` — copied from
`demos/_shared/portfolio/`, with `01.home` replaced by
`packages/theme-blox/demo-home/` (landing sections + collections). Theme
docs fold in at `02.theme-docs/`. Re-run `demo:link` after editing either
source. Theme is symlinked from `packages/theme-blox/`.
